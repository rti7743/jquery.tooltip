jquery.ui なんていらなくなったぜ!

jquery.ui の tooltip があまりにもアレだったので、tooltip2 として自作してみました。
ただ、jquery ui が嫌いな人もいるでしょうから、 jquery ui を使わないでも動作するバージョンも作成しました。
ご自由にお使いください。


download

jquery.ui を使わないバージョンをダウンロードする

利点.

・CSS だけで吹出しを実現

・みんな大好きIE6対応

・普通の吹出しから、
ボーダーを突き破る漫画のような吹出し、
ハートマークなどの心の動きみたいな吹出しまで大抵のものはサポート。
(border-radisの関係で 丸い吹出しだけサポートしていません。)

・吹出しにフォーカスを移動させることが可能
噴出しにリンクをおいて、そのリンクをクリックすることが出来るみたいな・・・

・噴出しの内容は端子なテキストから、任意のdiv要素までなんでもできる。
